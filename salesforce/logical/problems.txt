1. Two Sum
--------------------------------------------------------------------------------
Category    : Easy
Description : Program to return pair of elements from an array that sum up to a given number.
Solutions   : 
  - Double for loop (Time: O(n^2))
  - Negation logic / Hash Map (Time: O(n))
Notes       : 1st problem in Leetcode

2. Longest Substring Without Repeating Characters
--------------------------------------------------------------------------------
Category    : Medium
Description : Program to return longest substring in a given string without repeating chars.
Sample I/O  : 
  - i/p: abcabcbb        -> o/p: abc
  - i/p: abcdeafbdgcbb   -> o/p: eafbdgc
  - i/p: pwwkew          -> o/p: wke
Solutions   : Brute force, Divide and conquer, Sliding Window
Notes       : 3rd problem in Leetcode
